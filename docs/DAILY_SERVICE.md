# 일일 포트폴리오 서비스 (단계별)

하나만 고를 필요 없습니다. **매일 브리핑**을 중심으로 아래 순서로 확장합니다.

## ✅ 1단계 — 앱 일일 브리핑 (적용됨)

- 대시보드 최상단 **「일일 브리핑」** (매일 접속 시 확인)
- 전일 스냅샷 대비 손익·종목별 등락
- 시그널: 고점 대비 −7% / 평단 대비 +10% (설정 변경 가능)
- 타이밍선(매수·매도)과 함께 표시
- **리포트 텍스트 복사** → 메모·카톡·Cursor 채팅에 붙여넣기

> 전일 대비는 **전날 첫 접속 시 저장된 스냅샷** 기준입니다.  
> 고점 추적은 앱 사용 중 기록된 최고가 기준입니다.

## ✅ 2단계 — 시세·정보 (Phase B, 적용됨)

- **KIS 확장**: 전일가·등락률·고/저 (새로고침 시 저장)
- **모바일 Summary 카드**: 좁은 화면에서 카드, `md` 이상은 표
- **CSV 가져오기**: 종목 Summary 하단 · 한투/키움 체결 CSV
- **KOSPI 벤치마크**: 보유 가중 전일比 vs KOSPI · 초과(α)

## ✅ 3단계 — 전략·편의 (Phase C, 적용됨)

- **타이밍 % 설정**: 매수 1·2차 / 매도 1·2차 (일일 브리핑 → 전략·시그널 설정)
- **목표가 알림**: 종목 상세 → 목표가 입력 + 「브라우저 알림」 체크
- **기간별 리포트**: 월별·연별 실현손익
- **분할·배당**: 종목 상세 → 기업 이벤트 (분할 시 매매 자동 조정)
- **KIS 보정**: 전일가·KOSPI 등락 역산 (`normalizeQuote`)

## 🔜 4단계 — 확장 (Phase D)

- 시그널 %를 타이밍 레이더에도 반영
- **브라우저/이메일 알림** (목표가·−7% 도달)
- **월별·연별** 손익 리포트
- 분할·배당 이벤트

## 🔜 4단계 — 확장 (Phase D + Cursor)

- 다계좌 · 차트 · PER/업종
- **Cursor Automation**: 매일 아침 「포트폴리오 브리핑」+ 거시 뉴스 요약 (선택)
- 금·현금·대출 등 **전체 자산 Runway** (별도 모듈)

---

## 매일 쓰는 방법

1. Vercel URL 또는 `npm run dev` 접속
2. **KIS 새로고침** (설정된 경우)
3. **일일 브리핑** 펼쳐서 시그널 확인
4. 필요 시 **리포트 텍스트 복사**

## Cursor 채팅과 함께 쓰기

앱 숫자는 **정확한 원천**, Cursor는 **뉴스·해석** 보조:

> 「오늘 일일 브리핑 복사본 붙여넣기 + 반도체 섹터 이슈 요약해줘」

이렇게 하면 **매일 자동 숫자(앱) + AI 해석(채팅)** 조합이 됩니다.
