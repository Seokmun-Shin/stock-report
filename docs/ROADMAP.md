# 기능 로드맵

현재 버전과 데이터 호환을 유지하며 단계별 적용합니다.  
각 Phase는 Git 태그/브랜치로 복원 지점을 두는 것을 권장합니다.

| Phase | 상태 | 설명 |
|-------|------|------|
| **A** | 진행 중 | 정확성·신뢰 (평단·실현손익·수수료·모달·체결시각) |
| **B** | 예정 | 시세·정보 (KIS 확장·모바일 카드·CSV·벤치마크) |
| **C** | 예정 | 전략·편의 (타이밍 설정·알림·기간 리포트·분할/배당) |
| **D** | 예정 | 확장 (다계좌·차트·펀더멘털) |

## Phase A 체크리스트

- [x] 평단(비용 포함) + 평가수익률 %
- [x] 매매 행별 실현손익
- [x] 수수료/세금 프리셋 + 매도 세금 분리
- [x] 종목 편집 모달
- [x] 체결시각 (동일일 FIFO 순서)
